OCTAL

adjective · 2 syllables · /'ɑk.təl/

OCTAL

What does "OCTAL" mean?

Relating to a number system with base eight, using only the digits 0 through 7.

Meanings

  1. Of or denoting a numbering system that uses base eight and the digits 0 to 7. The old file permissions were written in octal, so 755 meant read-write-execute for the owner. technical
  2. The octal numbering system itself, or a number expressed in it. He converted the binary string into octal to make it easier to read. technical

Did you know?

  • Octal survives in everyday computing through Unix file permissions: a code like 644 is read in base eight, where each of the three digits packs three on-or-off bits into a single 0-to-7 number.

Word origin

From Latin 'octo' (eight) plus the adjective ending '-al'; coined to parallel 'decimal', for counting in groups of eight.

Remember it

OCTAL shares its 'OCT' with octopus and October - all built on the Latin 'octo', eight.
